Basic tensor algebra subprograms (BTAS)
********************

This chapter describes |projectName|'s support for tensor (multilinear) algebra algebra routines, 
such as the tensor contraction. These routines are group similar to routines in the Basic Linear
Algebra Subprograms (BLAS).
